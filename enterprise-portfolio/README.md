# 企業のポートフォリオ管理 — HST に基づく境界面観測

**正本（Canonical）**: 日本語版。

本フォルダは、**企業のポートフォリオ管理**に関する HST の適用を定義する。組織というブラックボックスの境界面観測により、過度な内部調査を避ける。

---

## 事業ポートフォリオの管理

組織というブラックボックスの系内部の構造を定義することは重要だが、現地に行って現場の人たちと会話するのはすればするほど系内部の散逸構造の罠にかかる。系の内部について知りたければ、表層に見える系のインターフェース—社長や管理職など—と会話すれば十分である。

---

## 投資デューデリジェンス

まずは期待される IRR を先に定義し、その IRR に対する資本の In/Out がハードルレートを超えるかどうかというクエリサンプルを持っていれば、過度に内部に拘泥することなく、市場における対象会社のランドスケープ上のトポロジーについてイベントホライズンを観測するだけで推測できる。買収後のパフォーマンス（予測される座標）を高い精度で推論することができる。
